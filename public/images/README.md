# Landing page images

Save your Gemini-generated images here with these EXACT filenames. They appear on the
homepage automatically (Next.js serves this folder at /images/...). Until a file
exists, the slot shows a warm colour block.

Export JPG, sRGB, ~2000px on the long edge.

| Filename | Where it shows | Aspect | Gemini prompt (see DESIGN-GUIDE.md) |
|---|---|---|---|
| hero.jpg | Hero, top right | Portrait 4:5 | #1 — framed dog portrait on a sunlit wall, negative space |
| product-framed.jpg | "Put them on everything" tile 1 | Square 1:1 | #3a — framed print in a styled room |
| product-print.jpg | tile 2 (Unframed print) | Square 1:1 | unframed rolled/flat art print leaning on a shelf, warm room |
| product-case.jpg | tile 3 (Phone case) | Square 1:1 | #3c — glossy phone case with pet portrait in hand |
| product-mug.jpg | tile 4 (Mug) | Square 1:1 | #3b — white mug with pet portrait on a wooden desk |

Keep the palette words in every prompt (bone, terracotta, sage, warm, editorial,
natural light) so the set looks cohesive. Square tiles crop centre, so keep the subject
centred.
